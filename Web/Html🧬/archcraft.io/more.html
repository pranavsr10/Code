Release Cycle

Like Arch Linux, Archcraft is also a rolling-release distribution. Which mean (After January 2023 release) there is no need to download a new ISO to update your existing system.
Existing Archcraft users can update their system anytime by running sudo pacman -Syyu

New ISO will be released Quarterly. Which mean any date in January, April, July and October.
Unless there are some major bugs or upstream issues, that requires immediate attension.
