What is Fall OS ?


Fall OS is just another arch-based Linux distribution. It uses i3wm and lightweight applications,
which makes it super fast and light-weight,and it runs out of the box.



Features

  • Minimal

    Fall OS is very minimal and lightweight.
    It consumes under 500Mb of memory as it only
    uses i3-window manager and some lightweight
    applications.

  • Stunning

    Flat and pastel colors, the consistency among
    all the UI elements,And various themes make
    Fall-OS look very clean, minimal and Stunning.

  • Powerful

    With built-in support for AUR, Chaotic AUR and
    being lightweight,Fall OS gives you the power to
    do whatever you want on your Linux PC.
